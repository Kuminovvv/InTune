# Interview Copilot

Локальный ассистент для подготовки и прохождения технических собеседований. Приложение полностью написано на Python и включает аудиопайплайн, локальную базу знаний и графический интерфейс на PySide6.

## Структура

```
app/
  python_core/   # аудиозахват, VAD, STT, RAG и работа с Ollama
  ui/            # PySide6-интерфейс (overlay, настройки, трей, хоткеи)
config.json      # конфигурация приложения
start.py         # автоматический запуск (.venv + GUI)
```

## Быстрый старт

1. Установите Python 3.10+.
2. Убедитесь, что установлен [Ollama](https://ollama.com/) и загружена модель `qwen2.5:7b-instruct` (однократно выполните `ollama run qwen2.5:7b-instruct`).
3. Запустите скрипт автоматического старта:

   ```bash
   python start.py
   ```

   Скрипт создаст виртуальное окружение `.venv`, установит все Python-зависимости (включая PySide6) и запустит полноценное приложение с оверлеем и настройками.

## Возможности

* **Аудио**: захват системного звука через WASAPI loopback, автоматическое переподключение при смене устройства, тестовый тон и измерение RMS/Peak для проверки захвата.
* **Речь**: VAD с регулируемой агрессивностью, распознавание через `faster-whisper`, частичные транскрипты в реальном времени.
* **RAG**: импорт файлов и папок в локальную базу (SQLite + FAISS), фильтр по профилю, переиндексация кнопкой `Импорт`.
* **LLM**: генерация подсказок через локальный Ollama (`qwen2.5:7b-instruct`) с режимами `default`, `shorten` и `code`.
* **PySide6 UI**:
  * overlay-окно (frameless + always-on-top) со статусом, последней транскрипцией и подсказкой;
  * окно Settings с вкладками Аудио, Речь, LLM, Оверлей, Знания и Профили;
  * системный трей для запуска/остановки прослушивания, открытия настроек и выхода;
  * глобальные хоткеи (`Ctrl+Space` — показать/скрыть оверлей, `Ctrl+R` — сжать подсказку, `Ctrl+Q` — запросить пример кода).

## Конфигурация

Все параметры хранит `config.json`: аудио-устройство, язык распознавания, модель Ollama, геометрию оверлея, настройки RAG и текущий профиль. Изменения из окна Settings мгновенно применяются и сохраняются в файл.

## Ограничения

* Код ориентирован на Windows (WASAPI loopback), однако большинство компонентов работает и на других ОС при наличии подходящего устройства вывода.
* Для импорта DOCX используется пакет `python-docx`; для PDF — `pypdf`.
* Работа глобальных хоткеев зависит от прав доступа библиотеки `keyboard` (на Linux может потребоваться запуск с sudo).

## Основные горячие клавиши

* `Ctrl+Space` — показать/скрыть overlay.
* `Ctrl+R` — запрос «сделай подсказку короче».
* `Ctrl+Q` — запрос «мини-пример кода».

## Лицензия

Проект предназначен для внутреннего использования и демонстрации офлайн-пайплайна собеседований.
